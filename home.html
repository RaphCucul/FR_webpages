Videoframes registration

The program is based on a console application written in C++ aimed on the registration of frames, using OpenCV library image processing functions. The source code of the console app with additional informations (in Czech) can be found here. Check the Wiki page to learn more about the program.

The program is designed to register frames with specific content - retinal fundus. The example of such videoframe is shown below.

The program is using Frangi filter during the frame preprocessing. The source code for Frangi filter can be found here. The program was created using OpenCV 3.2.0 and Qt 5.9.1. To run the program properly, it is necessary to modify phase correlation functions in the OpenCV source before compiling the library. Folder OpenCV_phaseCorr contains files with modified content.
